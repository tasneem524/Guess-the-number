# 🎯 Number Guessing Game

A fun and interactive terminal-based game built using Python.
The computer randomly selects a number between 1 and 100,
and you have 7 lives to guess it correctly!

## 🕹️ Features
- Random number generation using Python's random module
- Low/High hints to guide your guess
- Lives system — lose a life on every wrong guess
- Play again option after each round

## 🛠️ Tech Used
- Python 3
- random module (built-in)

## ▶️ How to Run
python guess_number.py
